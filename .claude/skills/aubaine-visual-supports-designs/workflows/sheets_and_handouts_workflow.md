# Workflow: sheets, cards, inserts, and handouts

## Character sheets

1. Identify the table actions the sheet supports.
2. Group fields by use during play, not by rulebook chapter.
3. Prioritize writable field size and pencil legibility.
4. Use labels outside fields when possible.
5. Keep calculated values near their inputs.
6. Provide a printer-friendly version without heavy backgrounds.
7. If digital-fillable, test tab order and field names.

## Reference cards

- One topic per card.
- Title in a stable place.
- Minimum body size appropriate to final card size.
- Bleed/safety/corner-radius allowance.
- Redundant color/icon/label coding.
- Cards remain understandable in grayscale.

## Handouts

- In-world handouts may be artful, but rules-critical text needs a readable version.
- If distressed type is used for mood, provide a clean transcription.
- Include GM-only and player-facing versions when secrets are present.

## Inserts and boxed-set papers

- Inventory each component.
- Define folding/cutting/assembly instructions.
- Keep bleed and safety for cut parts.
- Use unique component IDs or slugs for printer communication.
