# Content outline template

Use this as a starting point for book-like visual supports. For sheets, screens, handouts, and cards, use only the relevant sections and add the component templates from `standards/visual_supports_design.md`.

## Front matter / opening support

- Half title or cover/title panel, if useful
- Title page or title block
- Credits and copyright
- License and attribution
- Compatibility disclaimer
- Accessibility note
- Safety note, if content warrants it
- Table of contents or component index
- How to use this support
- Quickstart or first-session path

## Chapter/Section 1: What this is

- One-sentence pitch
- What players do
- What the GM/facilitator does
- What changes over a session/campaign
- Required materials
- What this artifact helps the table do

## Chapter/Section 2: Core procedure

- Core mechanic or primary use procedure
- Turn/session/support-use loop
- Example of play or use
- Common terms
- Quick reference box

## Chapter/Section 3: Characters, roles, or users

- Creation steps or setup steps
- Choices that matter immediately
- Starting resources or fields
- Advancement/change/update procedure
- Sheet/form references

## Chapter/Section 4: Play procedures

- Exploration / investigation / social / conflict procedures
- Risk, harm, recovery
- Equipment/resources
- Conditions/tags
- Table-facing summaries

## Chapter/Section 5: GM tools / facilitator tools

- Prep procedure
- Scene framing
- Faction moves
- Encounter procedure
- Consequences
- Random tables
- Reference dashboard content

## Chapter/Section 6: Setting/adventure material

- Locations
- Factions
- NPCs
- Threats
- Rewards
- Clocks/timers
- Clues and player-facing handouts

## Component appendix

- Character sheet
- Faction sheet
- GM screen panel list
- Reference cards
- Player handouts
- Trackers/clocks
- Forms and worksheets
- Home-print components

## Back matter

- Rules reference
- Tables appendix
- Glossary
- Index
- License notices
- Asset credits
- Errata/change log URL
- Colophon
