# Aubaine Content Writer Skill

A complete AI skill package for creating and revising tabletop roleplaying game material.

It is built to produce table-ready content with no AI tells: direct language, concrete details, usable procedures, complete mechanics, and strong adventure design.

## Contents

- `SKILL.md` — main skill instructions with YAML front matter
- `docs/best-practices.md` — documented TTRPG design principles and how to apply them
- `docs/no-ai-tells.md` — forbidden phrasing, cleanup rules, and replacement patterns
- `docs/system-profiles.md` — system-aware guidance for common design families
- `docs/safety-accessibility.md` — safety, consent, and accessibility practices
- `checklists/quality-gates.md` — final review gates
- `examples/` — before/after examples
- `bibliography.md` — source map and citations
- `manifest.json` — package metadata

## Installation

Use `SKILL.md` as the main skill file. Keep the supporting folders with it so the agent can consult the guidance docs, checklists, the style example, and references.

## Design Goal

The skill should make content that is:

- Playable immediately
- Clear under table pressure
- Respectful of player agency
- Mechanically complete
- System-aware
- Safe around mature themes
- Easy to scan
- Free of AI-sounding filler

## Core Rule

Do not write like a pitch. Write like a useful game text.
