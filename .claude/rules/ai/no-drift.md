# No drift

Do not restate in a file what already lives elsewhere. Reference the source; never copy it.

- No hardcoded counts, such as "four roll types" or "the six characteristics". A count breaks
  when the set changes.
- No file, folder, or reading-order lists in prose. The tree is the source, and there is no
  reading order to maintain.
- No copying a value another source owns, whether it is authored elsewhere or generated.

Describe the concept and let the reader find the members in their one source. Keep each authored
value in exactly one place, and prefer generated output over a hand-maintained copy.
