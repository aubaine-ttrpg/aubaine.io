# Wiki

Game-content documentation for **Aubaine**, the fiction-first tabletop RPG. Pages here are written for players, game masters, and system designers — the mechanics, the math, the design vocabulary that drives the game at the table.

## Scope

- Core mechanics (dice resolution, abilities and aptitudes, resources like HP / Energy / Memory).
- Skill design: the two families (Unleashed, Specific), energy economics, damage bands, tags, writing conventions.
- Character progression: ability / aptitude growth, skill acquisition, XP rules.
- Reference material: status effects, distance bands, damage types, targeting language.
- Lore, factions, and world-level design as it is set down.
